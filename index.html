Our Work

Geofencing API

Geofencing API provides functionality used to detect whether a coordinate is inside a circle or polygon. Accept batch requests.
Certificate Toolkit API

Certificate Toolkit contains tools necessary to generate self-signed certificates, inspect existing certificates, and verify chains.
Ticker Tracker App

Ticker Tracker is a journaling app for tracking stock, crypto, forex and options trades. Includes a calendar with US market holidays, and analytics based off your trading history.

Our Tech

Nodeworx software is written in NodeJS and .NET. We use the latest, most secure and performant packages in all our products. Our APIs are hosted on Railway and written in .NET. Our web apps are written in Nuxt 3.
